FansInTears App

FansInTears is a space for football fun and engagement.

Features:

  • List of upcoming and live matches.
  • Chat Rooms - select one of the live or upcoming matches and enter a telegram group with all the fans engaged in the selected match.
  • Telegram Media Bot - all match groups have FansInTearsMediaBot inside which can get latest media by hashtag from Instagram or Twitter. Want to get a photo from the Arsenal-WestHam derby? Just ask the bot for the latest media by hashtag "arswes".
  • Coverage - FansInTears supports matches from 6 major football leagues: England Premier League, France Ligue 1, Spain LaLiga, Italy Serie A, Germany Bundesliga, Russia Premier League.

We want this app to be a greate place for football fans engagement so we keep adding new features and leagues.
